## AI의 단기적 방향성
- https://www.youtube.com/watch?v=aoYy5JeS6fA
  + Apps SDK (ChatGPT와 직접적으로 외부 앱 연동시키기)
    - 비개발자용
  + Building Agents (Agent Kit)
    - GUI로 에이전트를 쉽게 만드는 기능
    - 비/반(No/Half)개발자용(Power Automate 수준의 난이도)
  + Coding Agent (Codex 등)
    - 개발자용. 코딩 도우미

## 오늘의 주제 - Building Agents
- Apps SDK는 Apps를 AI와 연동하는 목적이라, 직접 만들 일은 많지 않을 것임
- Coding Agent는 모두가 사용하고 있을 것으로 예상됨
- Building Agents에 대해 논해보자
